### Content

The Datatable allows to display several types of data.

| Data                | Usage                                                                                                                                          | Width behaviour        |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| **Text**            | Used to display the name of the item.                                                                                                          | Fixed or fluid         |
| **Multi-line text** | ?                                                                                                                                              | Fixed or fluid         |
| **Number**          | The content is always right justified                                                                                                          | Fixed or fluid         |
| **Badge**           | Status                                                                                                                                         | Adapted to the content |
| **Input**           | ?                                                                                                                                              | Fixed or fluid         |
| **Checkbox**        | Only one column per table, always in first position (on the left)                                                                              | Fixed                  |
| **Options**         | Only on column per table, always in last position (on the right)                                                                               | Fixed                  |
| **Icon**            | Action ?                                                                                                                                       | Fixed                  |
| **Link**            |                                                                                                                                                | Fixed or fluid         |
| **KPI**             | ?                                                                                                                                              | Adapted to the content |
| **Expand**          | Only one column per table, always after the checkbox column, in first position when no checkbox column/ always in first position (on the left) | Fixed                  |

## Variations

### Sizes

Each cells are available in 3 sizes.

| Size             | Value        |
| ---------------- | ------------ |
| Small            | 2.5mu (40px) |
| Medium (default) | 3mu (48px)   |
| Large            | 3.5mu (56px) |
